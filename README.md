# AI Translator & Critic

Веб-приложение на Flask для перевода текста с использованием AI и оценки качества перевода.

## Установка

1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Запуск

Перейдите в директорию `src` и запустите приложение:
```
cd src
python app.py
```

Приложение будет доступно по адресу `http://127.0.0.1:3000/` или через Codespaces URL на порту 3000.

## Структура проекта

- `src/app.py`: Основная логика приложения.
- `src/templates/index.html`: HTML шаблон интерфейса.
- `requirements.txt`: Зависимости Python.
